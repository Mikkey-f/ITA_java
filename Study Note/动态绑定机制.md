## 示例：

### 1.

```java
package com.poly.detail02;

public class test {
    public static void main(String[] args) {
        A a=new B();
        System.out.println(a.sum());//30
        System.out.println(a.sum1());//30
        //直接调用B类的方法
    }
}
class A{
    int i=10;
    public int sum(){
        return i+10;
    }
    public int sum1(){
        return get()+10;
    }
    public int get (){
        return i;
    }
}
class B extends A{
    int i=20;
    public int sum(){
        return i+10;
    }
    public int sum1(){
        return get()+10;
    }
    public int get (){
        return i;
    }
}
```

### 3.

```java
package com.poly.detail02;

public class test {
    public static void main(String[] args) {
        A a=new B();
        System.out.println(a.sum());//30
        System.out.println(a.sum1());//30
        //过程分析
        //B类中没有sum1方法，就去调用父类A中的sum1方法，sum1的返回值中又调用了get函数；但此时子类B和父类A中都有get函数
        //因为a的运行类型为B，所以会去调用B的get函数，get函数返回的是i，问题又来了，应该返回B中定义的i还是A中定义的i呢？
        //答案：调用B的i，因为get函数是在B中定义的，就调用B中定义的属性
    }
}
class A{
    int i=10;
    public int sum(){
        return i+10;
    }
    public int sum1(){
        return get()+10;
    }
    public int get (){
        return i;
    }
}
class B extends A{
    int i=20;
    public int sum(){
        return i+10;
    }
//    public int sum1(){
//        return get()+10;
//    }
    public int get (){
        return i;
    }
}
```
