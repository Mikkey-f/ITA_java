## 1.Api--------Calendar

```java
1.获取当前时间的日历对象 : public static Calendar getInstance();
2.获取日期对象:   public final Date getTime();
3.给日历设置日期对象:  public final setTime(Date date);
4.拿到时间毫秒值:  public long getTimeInMillis();
5.取日期中的某个字段信息:  public int get(int field);
6.修改日历的某个字段信息:  public void set(int field,int value);
7.为某个字段增加/减少指定的值:  public void add(int field,int amount);


//细节1:    月份范围:0-11,0实际上是1月。
//星期: 1 2 3 4 5 6 7  :1代表星期日.

//细节2:  Calendar是一个抽象类，不能直接用new,而是通过一个静态方法获取到子类对象。
```
