1.事件源：是一个产生事件的对象，比如按钮，窗口

2.事件:承载事件源状态改变的对象，比如当键盘事件，鼠标事件，窗口事件等等，会生成事件对象,该对象保存着当前事件的信息。

java.awt.event包和javax.swing.event包中定义了各种事件类型
